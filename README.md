# Kornelia & Cameron — Wedding Site Redesign

A redesigned version of the Kornelia & Cameron wedding website, built as a proof-of-concept by Joy Boo.

Original site: https://sayi.do/korneliaandcameron6966

## Design

- **Theme**: Autumnal Highlands (sage, copper, forest green, cream)
- **Typography**: Cormorant Garamond + Playfair Display + Inter
- **Features**: Countdown timer, timeline schedule, photo gallery, accordion FAQ, scroll animations
- **Bilingual**: Full English + Polish content preserved
- **Responsive**: Mobile-first design

## Deployment

Hosted as a static site on Render at `weddingsite.joyboo.ai`.
